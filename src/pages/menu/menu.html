侧滑菜单
输入属性
Attr Type Details
content any 菜单应该使用的内容元素的引用
enabled boolean 如果为true，则菜单已启用。默认 true.
id string An id for the menu.
persistent boolean 如果为true，菜单将保留在子页面上
side string 菜单应该放置哪一侧。默认“左”
swipeEnabled boolean 如果为true，则启用滑动菜单。默认为true
type string 菜单的滑动类型"overlay", "reveal", "push".
